如何把python文件打包成exe

刚入门python的同学可能会有有一些疑惑

比如：

怎样把自己的python作品分享到其他电脑，但在其他电脑上运行不会看到该程序的代码

把py文件打包成exe就阔以啦

打开cmd黑窗口

按下键盘上的WIN+R快捷键，调出运行窗口

在运行窗口中，输入：CMD，然后回车确认输入，

就会弹出一个窗口

这就是cmd命令行窗口，

（win10的cmd窗口大概长这样）

（win7的是这样）

下载插件

在你的黑窗口中输入以下代码下载pyinstaller插件

pip install pyinstaller -i https://pypi.douban.com/simple/

等待一会儿，

下载完成！

使用一些简单cmd命令打开文件夹

首先，

将你的py文件保存在一个指定的路径中

输入

cd 文件夹名

如我要打开文件夹“yetutu”

就要在cmd中输入

cd yetutu

进入

cd..

退出

直接输入某个电脑中磁盘名称就可打开（默认是C盘）

比如我想用cmd打开D盘就直接输入

D:

输入 dir 就可在cmd中查看该文件夹的目录

注意：文件名和文件夹名称要为英文

开始打包

用上面的语法打开需要打包的py文件所在的路径

在cmd中输入命令

pyinstaller -F -w 要打包的py文件名.py

如，py文件名为yetutu.py

输入

pyinstaller -F -w yetutu.py

就可以啦

耐心等待就打包完成啦

等等。。

如果要给exe文件添加图标要怎么弄嘞

把ico图标和py文件放在同一路径

然后输入命令行

pyinstaller -F -i 图标名称.ico 需要打包的py文件.py

如我的图标名称是yetutu.ico

在cmd输入

pyinstaller -F -i yetutu.ico yetutu.py

随便推荐一个充满ico的ico图标库：

https://www.easyicon.net/

打包完成后可在该py文件路径中发现一个文件夹

里边有一个exe格式文件

这就是你打包完成的文件啦

双击打开

结束

拜

轮到滑稽上场啦
