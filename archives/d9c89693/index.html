代码编辑器-Visual Studio Code

Visual Studio Code 是一款免费好用的代码编辑器，语言全，易使用

安装

右键“我的电脑”选择“属性”可在“系统”中查看电脑是32位还是64位

然后双击相应的安装包安装

使用

visualstudiocode怎么设置中文

刚安装的 Visual Studio Code ，默认显示的是英文

使用快捷键Ctrl + Shift + P，打开相关设置功能菜单，选择Configure Display Language

这时菜单变为两个en和Install additional languages，选择第二个Install additional languages

在左边弹出可安装相关的语言插件，选择中文简体，点击Install

接着进行安装，可以查看到右侧显示中文简体语言包信息

安装完毕后，提示需要重启工具，点击右下角的Restart Now进行重启

重启完毕后，应用原先显示为英文变为中文简体

下载语言

点击左侧的应用商店

在搜索栏输入你需要安装的语言插件，如“python”

选择并安装
